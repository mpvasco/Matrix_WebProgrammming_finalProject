Figma - design accessible to everyone.

  If you are going to create a web page for yourself and you are still not clear on what you want the best way to focus on designing it first, or if you are going to design it for a client, it is also preferable to design it first until the customer validates the layout. and then we can go to the code. It is much simpler to change a layout in the design than to keep making changes to the code until we are satisfied with the appearance of the page.

  Figma is my favorite online design tool. There is a free version and a paid enterprise version, but if you work with personal projects, the free version works perfectly.

Benefits:

  • ✅ It's online, you can access it from anywhere
  • ✅ Easy and intuitive to move
  • ✅ There is a wide variety of templates so you don't start from a blank page
  • ✅ You can simulate animations and navigation between screens
  • ✅ You can simulate on several different devices and know if your design meets android and iphone, for example
  • ✅ You can easily share your design with friends
  • ✅ You can have the css code of the components to help you when developing
  • ✅ It is a professional tool that adds a lot of value in the presentation of a project

GitHub - Let's build from here

  If you work on several projects on several different computers, or even as a team, gitHub can be an excellent option. You can store your codes for free in the cloud and access them from anywhere over the internet.

  Many devs also use github to create project portfolios and send the link to recruiters to see what devs have been working on.

  You can collaborate on opensource projects and people can collaborate on your projects.

  The best part of github is the code versioning, you don't need to keep creating copies or backups, you just need to create commits as points in the history and whenever you need you can recover the old versions.

  If you have private projects that you want to keep on github you can control whether the project is public or private.

Visual Studeo Code - from Microsoft

At its heart, Visual Studio Code features a lightning fast source code editor, perfect for day-to-day use. With support for hundreds of languages, VS Code helps you be instantly productive with syntax highlighting, bracket-matching, auto-indentation, box-selection, snippets, and more. Intuitive keyboard shortcuts, easy customization and community-contributed keyboard shortcut mappings let you navigate your code with ease.

For serious coding, you'll often benefit from tools with more code understanding than just blocks of text. Visual Studio Code includes built-in support for IntelliSense code completion, rich semantic code understanding and navigation, and code refactoring.

And when the coding gets tough, the tough get debugging. Debugging is often the one feature that developers miss most in a leaner coding experience, so we made it happen. Visual Studio Code includes an interactive debugger, so you can step through source code, inspect variables, view call stacks, and execute commands in the console.

VS Code also integrates with build and scripting tools to perform common tasks making everyday workflows faster. VS Code has support for Git so you can work with source control without leaving the editor including viewing pending changes diffs.
