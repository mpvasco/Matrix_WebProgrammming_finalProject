Figma - design accessible to everyone.

If you are going to create a web page for yourself and you are still not clear on what you want the best way to focus on designing it first, or if you are going to design it for a client, it is also preferable to design it first until the customer validates the layout. and then we can go to the code. It is much simpler to change a layout in the design than to keep making changes to the code until we are satisfied with the appearance of the page.

Figma is my favorite online design tool. There is a free version and a paid enterprise version, but if you work with personal projects, the free version works perfectly.

Benefits:

  • It's online, you can access it from anywhere
  • Easy and intuitive to move
  • There is a wide variety of templates so you don't start from a blank page
  • You can simulate animations and navigation between screens
  • You can simulate on several different devices and know if your design meets android and iphone, for example.
  • you can easily share your design with friends
  • You can have the css code of the components to help you when developing.
  • It is a professional tool that adds a lot of value in the presentation of a project.
